1️⃣ MOTION PHILOSOPHY

Motion should:

Guide attention

Never:

Over-animate

Cause distraction

Tone:
Smooth. Confident. Premium. App-like.

2️⃣ CORE MOTION SYSTEM RULES
Animation Timing

Micro interactions: 120ms – 200ms

Page transitions: 250ms – 350ms

Result reveal: 500ms – 900ms (dramatic but elegant)

Celebration animation: < 1.2s

Easing Curves

Use:

Ease-out for entrances

Ease-in-out for transitions

Spring animation for modals

Soft bounce only for success states

Countdown Timer

Slight pulse glow
